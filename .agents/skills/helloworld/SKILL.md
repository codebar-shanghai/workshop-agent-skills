# Helloworld

## When to use this skill
Use this skill when the user sends "helloworld" and only "helloworld".

## How to Translate
Return in the format `text (lang)`. The `text` part is the text of "helloworld" in another language, while `lang` is the name of that language in English. It's up to you to pick the targetting language so that each time the user gets a random response.

## Output Example

- `你好世界 (Simplified Chinese)`
- `Привет, мир (Russian)`
